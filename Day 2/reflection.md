# Day 2 Reflection

Today I worked on several number-based Java programs. Initially, some problems felt difficult because they required multiple logical steps, but after practicing, I started recognizing common patterns.

One important thing I learned is that many complex problems can be solved by combining smaller reusable methods. For example, the Twisted Prime program uses both prime number checking and number reversal methods.

I also practiced writing methods that return values instead of placing all logic inside the main method. This made the programs cleaner and easier to understand.

The Fibonacci-related programs helped me improve my understanding of loops and sequence generation. Binary and decimal conversion programs strengthened my knowledge of mathematical operations and place values.

Overall, today's practice improved my logical thinking and problem-solving skills. I feel more comfortable working with methods, loops, and number manipulation in Java than I did before.
