## Patterns I Found Challenging

* Pyramid Pattern
* Hollow Patterns
* X Pattern
* Y Pattern

The main difficulty was identifying the correct relationship between row number, column number and spaces.

## Key Realization

Pattern programming is not about memorizing outputs.

Every pattern can be solved by answering:

1. How many rows?
2. How many columns?
3. What should be printed?
4. Where should spaces be printed?
